【script.js(-3点)】

・if文でhasClassとremoveClassをつけるのも良いですが、もっとスマートに書く方法があります。※※これチャレンジ！！●
それがtoggleClassです。
toggleClassを使えば「クリックした時にクラスを付与し、もう一度クリックしたらその付与されたクラスも削除する」を簡単に実装できます。
toggleというのはJSでよく出てくる単語で、「切り替え」を意味しており、他にもslideToggleやtoggleなど似たような便利なものもあるので覚えておきましょう！

・appendでHTMLにスタイルを直接書き、.topを不透明にするのはあまりよくありませんね。(-2点)●
なので
    $('head').append(
        '<style type="text/css">.top{opacity:0;}</style>'
    );
はゴッソリ削除し、初めからCSSでopacityを0に指定しておきましょう！

・例えば下記のように変数を指定していますが
    var effect_pos = 250; // 画面下からどの位置でフェードさせるか(px)
    var effect_move = 30; // どのぐらい要素を動かすか(px)
    var effect_time = 1000; // エフェクトの時間(ms) 1秒なら1000
これを二重に指定している箇所がありますね。(-1点)●
CSSと同じくこれも下に書かれたもので上書きはされますが、できるだけ別々の名称をつけた方が良いでしょう！
var effect_time = 1000;
のように完全に一致しているものは2つ書かなくても良いですね。
